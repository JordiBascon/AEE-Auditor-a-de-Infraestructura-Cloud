# AEE.: Reto de Refactorización

### ¿Cúantas líneas de código habéis ahorrado al usar grupos?
En mi caso no he ahorrado líneas ya que la primera versión del XSD tiene `222 líneas` mientras que la segunda versión del xsd tiene `255 líneas`, solo hemos despejado el código y hacerlo más legible y mas accesible a cambios en un futuro.

### ¿Qué error os da VS Code si intentáis poner dos servidores con el mismo ID?
Da el error `cvc-identity-constraint.4.1: Duplicate unique value declared for identity constraint of element 'id'.`
